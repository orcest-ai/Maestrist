# راهنمای API و زیرساخت برای Maestrist

## تنظیمات پیشنهادی در Maestrist

در **Settings → LLM** این مقادیر را تنظیم کنید:

- **Model:** `gpt-4o` یا `claude-sonnet-4-20250514`
- **Temperature:** 0 (برای کار دقیق فنی)
- **Max iterations:** 500

## ۱. بهترین مدل‌ها برای اتوماسیون (مشابه Devin / ChatGPT Pro Agent)

| مدل | بهترین کاربرد | قیمت تقریبی (۱M توکن) |
|-----|---------------|------------------------|
| **GPT-4o** | تعادل عالی کدنویسی و درک | $2.50 ورودی / $10 خروجی |
| **Claude Sonnet 4** | بهترین قیمت/کیفیت برای کد | $3 ورودی / $15 خروجی |
| **Claude Opus 4.1** | حداکثر کیفیت برای کارهای پیچیده | $15 ورودی / $75 خروجی |
| **GPT-4o-mini** | ارزان برای تست و کار سبک | $0.15 ورودی / $0.60 خروجی |

**پیشنهاد پیش‌فرض:** `gpt-4o` یا `claude-sonnet-4-20250514` برای اتوماسیون‌های فنی، تست سرور، و توسعه نرم‌افزار.

## ۲. بهترین سرویس‌های API پریمیوم

### OpenAI (پیشنهاد اصلی)
- **صفحه:** https://platform.openai.com
- **مدل‌ها:** GPT-4o, GPT-4o-mini, o1
- **محبوب‌ترین** برای agentها و کدنویسی

### Anthropic Claude
- **صفحه:** https://console.anthropic.com
- **مدل‌ها:** Claude Sonnet, Claude Opus
- **مزیت:** کیفیت بالا، پشتیبانی عالی از کد و زبان فارسی

### Google Gemini
- **صفحه:** https://aistudio.google.com
- **مدل‌ها:** Gemini 2.0, Gemini Pro
- **قیمت رقابتی**

---

## ۳. Hugging Face Pro

- **قیمت:** حدود $9/ماه
- **محتویات:** $2 اعتبار رایگان ماهانه + استفاده بدون نرخ اضافی
- **مدل‌های پیشنهادی:** Llama 3, Mixtral, Code Llama
- **محدودیت:** برای آزمایش مناسب، نه برای production سنگین
- **لینک:** https://huggingface.co/pricing

### پیشنهاد اقتصادی (HF Pro)
- Code Llama 34B برای کدنویسی
- Mixtral 8x7B برای کارهای عمومی

### پیشنهاد با هزینه بالاتر (HF)
- Inference Endpoints با مدل‌های اختصاصی
- سرور مجازی روی HF برای مدل‌های بزرگ

---

## ۴. هاستینگ سرور شخصی

### DigitalOcean
- **پیشنهاد:** Droplet با ۴GB+ RAM
- **قیمت:** از حدود $24/ماه
- **مناسب برای:** اجرای Maestrist + Docker + مدل‌های کوچک محلی
- **لینک:** https://www.digitalocean.com

### سرور در ایران (پیشنهاد کلی)
- **نیاز:** CPU قوی، RAM حداقل ۱۶GB، بدون محدودیت Docker
- **نکته:** برای اتصال به APIهای خارجی، اتصال پایدار اینترنت ضروری است
- **پیشنهاد:** VPS با دسترسی root از ارائه‌دهندگان معتبر داخلی
- **مدل پیشنهادی:** استفاده از API ابری (OpenAI/Anthropic) روی سرور ایرانی؛ اجرای مدل محلی تنها در صورت GPU مناسب
